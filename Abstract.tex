
\begin{abstract}
Unreal Engine is an popular video game development engine which in recent years has gained a lot of popularity in various different fields for all of the functionalities and features it offers. But those aren't enough to fulfil all the new requirements expected from it, especially as some would generally not be used in game development. One such feature is importing and creating new 3D objects from CAD files during the runtime of an Unreal program. In this thesis it will be demonstrated how this can be achieved and how users can interact with these new objects. For these purposes a plug-in for Unreal Engine and a standalone, interactive program were developed.
\end{abstract}

\begin{abstract}
Die Unreal Engine ist eine beliebte Engine für die Entwicklung von Videospielen, die in den letzten Jahren aufgrund ihrer zahlreichen Funktionen und Merkmale in verschiedenen Bereichen sehr beliebt geworden ist. Diese reichen jedoch nicht aus, um alle neuen Anforderungen zu erfüllen, die an sie gestellt werden. Zumal einige davon in der Regel nicht für die Spieleentwicklung verwendet werden. Ein solches Feature ist das Importieren und Erstellen neuer 3D-Objekte aus CAD-Dateien während der Laufzeit eines Unreal-Programms. In dieser Arbeit soll gezeigt werden, wie dies erreicht werden kann und wie der Benutzer mit diesen neuen Objekten interagieren kann. Zu diesem Zweck wurden ein Plug-in für die Unreal Engine und ein eigenständiges, interaktives Programm entwickelt.
\end{abstract}
